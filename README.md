# 4dcelltrack
Open-source pipeline for 4D cell tracking and migration analysis in iPSC-derived assembloid live imaging.

Functions are provided to assist in the following:
<ol>
    <li><strong>Preprocessing</strong> - Convert raw microscopy data (.nd2) into trackable TIFF stacks.</li>
    <li><strong>Manual Tracking (External)</strong> - Perform cell tracking in ImageJ or similar software.</li>
    <li><strong>Data Collection</strong> - Aggregate tracking outputs into a unified dataset.</li>
    <li><strong>Filtering (Optional)</strong> - Remove duplicate or overlapping tracks.</li>
    <li><strong>Metric Computation</strong> - Calculate speed and displacement metrics per cell.</li>
    <li><strong>Statistical Analysis</strong> - Compare experimental groups and compute statistics.</li>
    <li><strong>Additional Metrics</strong> - Compute MSD and turning angles for deeper motion analysis.</li>
    <li><strong>Visualization</strong> - Generate representative trajectory plots.</li>
</ul>
